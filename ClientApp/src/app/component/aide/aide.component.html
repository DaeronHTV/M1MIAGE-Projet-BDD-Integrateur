Comment créer un compte VisioPAd?

Pour créer un compte VisioPAd, il vous suffit de vous rendre sur le formulaire de création de compte. Suite à cela, nous vous demanderons votre nom, prénom, adresse e-mail, ainsi que le nom de l'établissement Ehpad et le nom du résident dont vous souhaitez être en contact. Une demande de création de compte sera alors envoyée à l'établissement concerné, qui validera ou non votre compte.

Comment prendre un rendez-vous?

Pour prendre rendez-vous, il vous suffit de vous rendre sur la page de réservation de rendez-vous. Suite à cela, nous vous proposerons de choisir un résident parmis ceux rattachés à votre compte, puis de choisir un créneau parmi les diposnibilités proposées. Si des invités ont prévus de se joindre à vous pour la visioconférence, il vous suffira d'indiquer leurs noms, prénoms et adresses emails, afin que nous puissions leurs envoyer un mail de confirmation. Pour finir, il vous faudra valider votre demande afin que la réservation du ou des rendez-vous soient envoyée à l'établissement pour validation ou refus.

Comment annuler une réservation / un rendez-vous?

Pour annuler un rendez-vous, il vous suffit de vous rendre sur la page "Mes Rendez-vous". Cherchez le rendez-vous concerné puis cliquer sur "Annuler une réservation" ou "Annuler un rendez-vous" (cela dépend de l'avancée de votre réservation). Vous devrez alors confirmez l'annulation, puis vous recevrez un mail de confirmation d'annulation. À savoir: si des invités sont prévus, alors le rendez-vous est annulé seulement pour vous, il reste donc réservé pour les invités avec le/la résident(e) concerné.
